# Bakshy Top500 Ideological alignment of 500 domains based on facebook data

Bakshy Top500 Ideological alignment of 500 domains based on facebook
data

## Usage

``` r
bakshy
```

## Format

An object of class `data.table` (inherits from `data.frame`) with 500
rows and 7 columns.

## References

Bakshy, Eytan, Solomon Messing, and Lada A. Adamic. "Exposure to
ideologically diverse news and opinion on Facebook." Science 348.6239
(2015): 1130-1132.
