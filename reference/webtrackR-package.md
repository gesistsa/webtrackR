# webtrackR: Preprocessing and Analyzing Web Tracking Data

Data structures and methods to work with web tracking data. The
functions cover data preprocessing steps, enriching web tracking data
with external information and methods for the analysis of digital
behavior as used in several academic papers (e.g., Clemm von Hohenberg
et al., 2023
[doi:10.17605/OSF.IO/M3U9P](https://doi.org/10.17605/OSF.IO/M3U9P) ;
Stier et al., 2022
[doi:10.1017/S0003055421001222](https://doi.org/10.1017/S0003055421001222)
).
